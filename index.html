RestAssured API Framework - [Github]

This project is a Java-based API testing framework built using Rest Assured and Maven. It supports BDD-style feature files and uses a modular approach to build test data and perform validations. The framework is designed to be easily extendable and maintainable, allowing for quick adaptation to new API endpoints.

Key Features:

  • Modular architecture for easy maintenance and scalability
  • Supports BDD-style feature files for clear test case definitions
  • Extensive use of Rest Assured for API testing
  • Integration with Maven for build management
  • Comprehensive documentation and examples for quick setup

Tech Stack: Java, Rest Assured, Maven, BDD

For more details, check the README file in the repository.

Ecommerce API Framework (using AI) - [Github]

This project is an AI-assisted automated API testing framework for the [Demoblaze API] and related Ecommerce endpoints. It was designed and implemented with the help of AI-driven code generation and optimization, making the development process faster, more accurate, and highly maintainable.

By leveraging AI during development, the framework:

  • Automatically generated POJO classes from API payloads.
  • Suggested optimal TestNG and Rest Assured structures.
  • Ensured high code coverage and clean architecture.
  • Created data-driven test cases with dynamic assertions to handle varying API responses (JSON & plain text).
  • Reduced manual coding time while maintaining clean, modular architecture
  • Tech stack: Java, Rest Assured, TestNG, POJO Classes, AI-powered design assistance

For more details, check the README file in the repository.
